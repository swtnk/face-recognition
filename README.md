# Face detection and recognition
## Requirement
1. Python 3.10

## Setup First time
1. Install pipenv
2. python3 face_recognition.py setup
3. pipenv install --dev

## Reset Project
1. python3 face_recognition.py reset

## Clean Project
1. python3 face_recognition.py clean

## Run
1. python3 face_recognition.py \<action\>\
    action:
    1. setup
    2. clean
    3. reset
    4. add
    5. train
    6. recognize
    7. recognize
    8. recognize
    9. -h or --help
    10. -v or --version
